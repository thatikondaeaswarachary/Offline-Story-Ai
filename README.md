# Mythweaver: Offline Story AI

Mythweaver is a high-performance, fully offline AI storytelling application for Android. It utilizes local, on-device AI models to generate rich, creative stories without requiring an internet connection. 

## Features
- **100% Offline AI:** Runs entirely on your device for complete privacy and availability anywhere.
- **Multi-Language Support:** Native generation support for English, Telugu, and Hindi via specialized system prompts.
- **Dynamic Voice Narration:** Integrated multi-voice Text-to-Speech (TTS) with Google Translate fallback for missing local voices.
- **High-Performance Execution:** Ultra-fast output modes optimized for mobile hardware.

## Tech Stack
- Frontend: React + Vite + Capacitor for Android building
- Backend / Local Server: Node.js (Local Host) / Python (Model Management)
